About

SmartUnderline is an open-source JavaScript library which automatically adds descender-aware underlines to links. It can be downloaded on GitHub or installed for free on the Eager App Store.

Notes

While SmartUnderline works great on blogs and in other controlled-text enironments, there are a few things you should be aware of when considering using SmartUnderline on your website.

  • It requires that the text be on top of a solid background color.
  • It makes an assumption about selection color, which is OS- and browser-dependent.
  • Each smart-underlined link has 12 text-shadows drawn underneath it, which could potentially have performance penalties on mobile or older devices.
