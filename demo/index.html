About

SmartUnderline is an open-source JavaScript library which automatically adds descender-aware underlines to links. It can be downloaded on GitHub or installed for free on the Eager App Store.

Known limitations

While SmartUnderline works great on blogs and in other controlled-text enironments, it has performance considerations and limitations which prevent it from being an all-purpose link styling option.

  • It requires that the text be on top of a solid background color.
  • It has performance considerations due to its heavy use of text-shadow.
  • It makes an assumption about selection color, which is OS- and browser-dependent.
